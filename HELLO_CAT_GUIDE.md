# 🐱 Hello Cat - AI编程助手使用指南

## 📋 概览

Hello Cat 是一款专为开发者设计的智能AI编程助手插件，完美集成到 IntelliJ IDEA
开发环境中。它不仅仅是一个聊天工具，更是一位随时待命的编程伙伴，能够帮助你解答技术问题、解释代码概念、提供编程建议，甚至在你感到疲惫时给予励志鼓励。

## ✨ 核心特性

### 🤖 先进的AI集成

- **实时流式响应**：带有自然打字动画的流畅对话体验
- **多供应商支持**：兼容 OpenAI GPT 模型、Claude 以及自定义端点
- **智能回退系统**：当AI服务不可用时自动切换到备用方案
- **可配置的API密钥和模型选择**：提供最大的灵活性

### 🎨 专业UI设计

- **优化的聊天界面**：最大化开发效率
- **完美消息对齐**：用户消息右对齐，AI回复左对齐
- **圆角气泡设计**：内容自适应的精美视觉效果
- **自动滚动**：确保对话体验流畅自然
- **等宽字体优化**：提升代码可读性

### 💻 代码智能聚焦

- **智能代码块检测**：自动识别并格式化代码片段
- **行内代码高亮**：增强代码展示效果
- **Markdown支持**：支持标题、列表、粗体斜体等格式
- **特殊标记**：为不同类型的内容提供视觉分隔

### ⚡ 开发者体验优化

- **快捷键支持** (Ctrl+Alt+K)：快速访问插件界面
- **持久侧边栏标签**：保持持续对话不中断
- **工具栏集成**：直观的控件操作

## 🚀 快速开始

### 安装Hello Cat

#### 从JetBrains市场安装（推荐）

1. 在 IntelliJ IDEA 中：**File → Settings → Plugins**
2. 在市场(Marketplace)选项卡中搜索 "Hello Cat"
3. 点击 **Install** 并重启IDE

#### 手动安装

1. 从 [Releases](https://github.com/MMCISAGOODMAN/ai-cat-plugin/releases) 下载最新的 JAR 文件
2. 在 IntelliJ IDEA 中：**File → Settings → Plugins → Install Plugin from Disk**
3. 选择下载的 JAR 文件
4. 重启 IntelliJ IDEA

### 基本使用

#### 启动界面

1. **切换界面**：使用 **Ctrl+Alt+K** 快捷键或 **Tools → Hello Cat**
2. **调整窗口位置**：拖动猫窗口到你喜欢的位置
3. **开始对话**：在输入框中键入你的问题
4. **获取励志语录**：点击"Quote"按钮获得灵感

#### 聊天能力

#### 键盘快捷键

| 快捷键          | 功能               |
|--------------|------------------|
| `Ctrl+Alt+K` | 切换 Hello Cat 可见性 |

## ⚙️ 详细配置指南

### AI服务设置

1. 前往 **Tools → Hello Cat Settings**
2. 输入你的API密钥（AI功能必需）
3. 配置AI端点URL：
    - **OpenAI**: `https://api.openai.com/v1/chat/completions`
    - **自定义端点**: 必须遵循OpenAI兼容API格式
4. 选择你偏好的模型名称
5. 点击 **Save Settings**

### 插件功能配置

| 功能                       | 描述         | 默认值  |
|--------------------------|------------|------|
| Enable AI Cat            | 启用主要插件功能   | 已启用  |
| Auto-hide when inactive  | 5秒无操作后自动隐藏 | 已启用  |
| Show motivational quotes | 显示励志消息     | 已启用  |
| Quote interval (minutes) | 励志语录频率     | 30分钟 |
| Chat history size        | 存储的最大对话消息数 | 10条  |

## 🔧 故障排除

### 常见问题及解决方案

#### **API连接失败**

- 验证API密钥是否正确且有足够余额
- 检查端点URL是否能从你的网络访问
- 确认防火墙设置允许出站HTTPS连接
- 先用简单的curl命令测试API端点

#### **Cat界面未出现**

- 在 **Settings → Plugins** 中确认插件已启用
- 检查Hello Cat Settings中"Enable AI Cat"是否勾选
- 完全重启IntelliJ IDEA
- 临时禁用其他插件检查冲突

#### **流式动画不工作**

- 确保配置了有效的AI凭据
- 检查网络连接和代理设置
- 验证AI服务支持流式响应
- 尝试使用更简单的模型（如gpt-3.5-turbo）

#### **自动隐藏功能异常**

- 确认设置中"Auto-hide when inactive"已启用
- 检查鼠标移动追踪是否正常工作
- 验证没有其他应用程序拦截鼠标事件
- 尝试调整自动隐藏延迟设置

### 获取帮助

如果你遇到持续性问题：

1. **查看错误日志**：Help → Show Log in Finder/Explorer
2. **验证配置**：仔细检查Hello Cat Settings中的所有设置
3. **网络诊断**：手动测试API端点可访问性
4. **插件兼容性**：临时禁用其他插件测试
